## 1. Introduction

This Privacy Policy describes how the **Affiliate Product Showcase** WordPress plugin ("the Plugin") collects, uses, and protects information when installed on your WordPress website ("the Site").

The Plugin is designed with **privacy as a fundamental principle**. We believe in transparency and giving users control over their data.

---

## 2. Data Collection

### 2.1 Data We DO NOT Collect

The Affiliate Product Showcase plugin:

- ❌ **Does NOT collect personal information** from your website visitors
- ❌ **Does NOT track user behavior** or browsing patterns
- ❌ **Does NOT use cookies** for tracking or analytics
- ❌ **Does NOT send any data** to third-party servers
- ❌ **Does NOT perform analytics** or collect usage statistics
- ❌ **Does NOT implement phone-home** functionality
- ❌ **Does NOT check for updates** through external services (uses WordPress core)
- ❌ **Does NOT use CDNs** or external content delivery networks
- ❌ **Does NOT load external fonts** or icon libraries

### 2.2 Data We DO Store

The Plugin stores the following data locally on your WordPress database:

| Data Type | Description | Storage Location | Retention |
|-----------|-------------|------------------|-----------|
| Product Information | Product names, descriptions, prices | WordPress Database (`wp_posts`, `wp_postmeta`) | Until deleted by admin |
| Product Images | Uploaded product images | WordPress Media Library (`wp_content/uploads`) | Until deleted by admin |
| Settings | Plugin configuration settings | WordPress Database (`wp_options`) | Until changed or deleted |
| Categories/Tags | Product categorization data | WordPress Database (`wp_terms`, `wp_term_taxonomy`) | Until deleted by admin |
| Affiliate Links | Affiliate product URLs | WordPress Database (`wp_postmeta`) | Until deleted by admin |

### 2.3 Data We DO Process

The Plugin processes the following data locally:

- **Product Data:** Display and organization of affiliate products
- **Settings Data:** Configuration of plugin behavior and appearance
- **User-Generated Data:** Product information entered by administrators

All processing happens **locally on your server**. No data is transmitted externally.

## 3. External Connections

### 3.1 No External Connections

The Affiliate Product Showcase plugin makes **NO external connections**:

- ✅ No CDN usage (all assets are bundled locally)
- ✅ No external APIs
- ✅ No external font services
- ✅ No external icon libraries
- ✅ No analytics or tracking services
- ✅ No phone-home mechanisms
- ✅ No update check servers (uses WordPress core's built-in system)
- ✅ No telemetry or usage data collection

### 3.2 Affiliate Links

The Plugin **does display affiliate links** that redirect users to external merchant websites when clicked. This is the **only** external connection initiated by the Plugin.

**Important:** The Plugin:
- Does NOT track clicks on affiliate links
- Does NOT collect analytics on link clicks
- Does NOT share user data with affiliate merchants
- Simply redirects users to the merchant's website when they click a link

Affiliate merchants may have their own privacy policies that apply once a user clicks through to their website.
