# Phase Templates

This directory contains reusable templates for the project.

## Available Templates

| Template | Use Case |
|----------|----------|
| `adr-template.md` | Architecture Decision Records |
| `sprint-template.md` | Sprint planning and tracking |
| `bug-report-template.md` | Bug reports |
| `pr-template.md` | Pull requests |
| `meeting-notes-template.md` | Meeting notes |

## How to Use

1. Copy the template file to the appropriate location
2. Rename it appropriately
3. Fill in all sections
4. Remove any sections that don't apply

## Template Guidelines

### ADR Template
- Use for significant architectural decisions
- Number sequentially (ADR-001, ADR-002, etc.)
- Store in `/docs/adrs/`

### Sprint Template
- Create one per sprint
- Update daily during standups
- Archive after sprint retrospective

### Bug Report Template
- Use for all bug reports
- Include as much detail as possible
- Attach screenshots/videos for UI bugs

### PR Template
- Auto-populated when creating PRs
- Fill in all relevant sections
- Check all applicable checkboxes

### Meeting Notes Template
- Use for all team meetings
- Distribute within 24 hours
- Track action items to completion
