# Affiliate Product Showcase

## Development

### Plan Management

The project roadmap lives in `plan/plan_source.md`. Generated files (`plan_sync.md`, `plan_sync_todo.md`) are auto-generated - never edit manually.

**To update the plan:**
- Edit `plan/plan_source.md`
- Run `./plan/update-plan.sh` (Unix) or `\.\plan\update-plan.ps1` (Windows)
- Commit changes

See [plan/PLAN_WORKFLOW.md](plan/PLAN_WORKFLOW.md) for full workflow details.
